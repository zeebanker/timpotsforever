Accessibility Statement

Accessibility at Timpots Forever

Effective May 13, 2026

Our Commitment

Timpots Forever is committed to ensuring that our website is accessible to all users, including those with disabilities. We strive to meet Web Content Accessibility Guidelines (WCAG) 2.1 Level AA standards and continuously work to improve the accessibility of our digital experience.

Accessibility Features

Timpots Forever includes the following accessibility features:

  • Semantic HTML: The site uses proper HTML structure for screen reader compatibility
  • Keyboard Navigation: All interactive elements are accessible via keyboard
  • Color Contrast: Text and interactive elements meet minimum contrast ratio standards
  • Responsive Design: The site is fully responsive and works on all device sizes
  • Form Labels: All form fields are properly labeled for screen readers
  • Alt Text: Images include descriptive alt text where applicable
  • Skip Links: Navigation elements can be skipped for keyboard users
  • Font Sizing: Text can be resized without loss of functionality

Screen Reader Compatibility

Timpots Forever is compatible with major screen readers including:

  • NVDA (Windows)
  • JAWS (Windows)
  • VoiceOver (macOS and iOS)
  • TalkBack (Android)

Browser and Assistive Technology Support

The site is optimized for the following browsers and assistive technologies:

  • Chrome (latest versions)
  • Firefox (latest versions)
  • Safari (latest versions)
  • Edge (latest versions)

Known Accessibility Limitations

We are aware of and actively working to address the following:

  • Interactive Maps: The Alumni Map uses Leaflet.js, which has limited keyboard accessibility for individual markers. We are exploring solutions to improve this.
  • Third-Party Services: Supabase authentication and Cloudflare services may have their own accessibility limitations beyond our control.

Accessibility Settings

You can use your browser and operating system settings to customize your experience:

  • High Contrast Mode: Enable your system's high contrast setting for better visibility
  • Text Size: Most browsers allow you to zoom or change text size (Ctrl/Cmd + Plus/Minus)
  • Dark Mode: Use your browser or system dark mode settings
  • Screen Reader Settings: Configure your screen reader to read content as needed

Report Accessibility Issues

If you encounter any accessibility barriers while using Timpots Forever, we want to know. Please contact us with:

  • The page or feature where you encountered the issue
  • A description of the accessibility problem
  • Your browser and assistive technology (if applicable)
  • Steps to reproduce the issue

Email us at: admin@timpotsforever.org

We will respond to accessibility concerns within 5 business days and work toward a resolution.

Accessibility Resources

For more information about web accessibility, visit:

  • W3C Web Accessibility Initiative (WAI)
  • WCAG 2.1 Quick Reference
  • WebAIM Accessibility Resources

Third-Party Content

While we strive for accessibility, some third-party services and integrations may not meet accessibility standards. These include:

  • Supabase Authentication: Login and signup forms from Supabase
  • Cloudflare Services: Email routing and CDN services

We are actively exploring ways to improve accessibility for all third-party integrations.

Continuous Improvement

Accessibility is an ongoing commitment. We regularly test our site with accessibility tools and user feedback to identify and fix issues. Thank you for helping us make Timpots Forever accessible to everyone.
